# Projects presentation website in Vitepress

## Build the project

At the root of the project, run the following command:
::: code-group

```bash [npm]
npm run docs:build
```

```bash [yarn]
yarn docs:build
```

:::

## Run the project

At the root of the project, run the following command:
::: code-group

```bash [npm]
npm run docs:dev
```

```bash [yarn]
yarn docs:dev
```

:::
